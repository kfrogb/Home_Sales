# Home_Sales
Repo for Home Sales (Mod 22) homework

Project Title: Home Sales - PySpark Challenge
Project Description: In this challenge, you'll use your knowledge of SparkSQL to determine key metrics about home sales data. Then you'll use Spark to create temporary views, partition the data, cache and uncache a temporary table, and verify that the table has been uncached.

Using Code: Use Google Colab to evaluate data.

How to Use Project: Use Google Colab.
